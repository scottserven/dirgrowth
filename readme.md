# DirGrowth
CLI app to show the growth of a specified folder (and sub-folders) over time, based
on the last modified date of the file.

```shell
2021-03: ==========================                                             2.1 TB
2021-04: ================================                                       2.6 TB
2021-05: ==================================                                     2.8 TB
2021-06: =====================================                                  3 TB
2021-07: ======================================                                 3.1 TB
2021-08: ===========================================                            3.5 TB
2021-09: ===========================================                            3.5 TB
2021-10: ================================================                       3.9 TB
2021-11: ==================================================                     4.1 TB
2021-12: ====================================================                   4.2 TB
2022-01: =========================================================              4.6 TB
2022-02: =========================================================              4.7 TB
2022-03: ================================================================       5.2 TB
2022-04: ====================================================================   5.5 TB
2022-05: =====================================================================  5.6 TB
```
